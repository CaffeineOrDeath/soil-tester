(void*) irq_handler()

irq_handler will be passed when certain parts of the circuit are activated. For example, using a contact switch. When it is depressed it will activate a pin. When that pin is activted, it will send an IRQ to stop the current process and return with a callback function.

Parameters
(void*)irq_vector The _VECTOR being called
(void*)callback Callback function before returning
